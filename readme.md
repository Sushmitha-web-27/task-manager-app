# Task Manager App

## Features

- User Registration & Login
- JWT Authentication
- Create Tasks
- Edit Tasks
- Delete Tasks
- Search Tasks
- Task Status Management
  - Todo
  - In Progress
  - Done

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

## Assumptions

- Simple authentication flow used
- Tasks are managed after login
- Minimal and clean UI focused on usability

---

## Tradeoffs

- No drag and drop functionality
- Basic JWT implementation
- No advanced global state management library

---

## Technical Decisions

- Used MERN stack for full-stack implementation
- MongoDB Atlas used for cloud database
- REST APIs built using Express.js
- Tailwind CSS used for responsive UI styling
